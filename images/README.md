## Tips
- Shoot in good natural light or bright indoor lighting — avoid flash shadows
- Clean the furniture before shooting — remove dust, stickers, tape
- Include a person in some shots to show scale (optional but helps)
- JPEGs at 80–85% quality are fine — no need for giant files
- Compress images before uploading: use squoosh.app (free, browser-based)
- Rename files exactly as listed above — the website references these exact names
